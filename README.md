<h1><strong>React Testing Library - Testes da Pokédex</strong></h1>

<h2><strong>Descrição</strong></h2>
<p align="justify">
  Neste projeto, foram desenvolvidos testes automatizados para uma aplicação React previamente construída e configurada com Jest e React Testing Library. 
  A aplicação implementa uma Pokédex completa, e o objetivo deste exercício foi criar testes robustos que garantissem a corretude dos componentes sem a necessidade de alterar a implementação original.
</p>

<h2><strong>Funcionalidades Testadas</strong></h2>
<ul>
  <li align="justify">Renderização correta dos componentes da Pokédex.</li>
  <li align="justify">Testes de navegação e alternância entre Pokémon.</li>
  <li align="justify">Simulação de eventos (cliques, navegação, filtros).</li>
  <li align="justify">Testes assíncronos com uso de <code>findBy*</code>.</li>
  <li align="justify">Validação de elementos, textos e estados esperados.</li>
  <li align="justify">Testes de inputs, filtros e detalhes dos Pokémon.</li>
  <li align="justify">Cobertura de componentes reutilizáveis e testáveis.</li>
</ul>

  <h2><strong>Description</strong></h2>
  <p align="justify">
    This project focuses on writing automated tests for a fully implemented React Pokédex application. 
    All tests were written using Jest and React Testing Library, ensuring component correctness without modifying the original implementation.
  </p>

  <h2><strong>Features Tested</strong></h2>
  <ul>
    <li>Correct rendering of Pokédex components.</li>
    <li>User navigation and Pokémon switching.</li>
    <li>Event simulation (clicks, filters, navigation).</li>
    <li>Async flows tested with <code>findBy*</code> queries.</li>
    <li>Input and state validation.</li>
    <li>Reusable and refactor-safe test suite.</li>
  </ul>
